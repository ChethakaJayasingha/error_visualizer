### Running the Application

#### Install Dependancies

```sh
npm i
```

#### Start the Frontend Server

```sh
# Inside directory (new terminal)
npm run dev
```

- Client will run on: http://localhost:5173
- Open your browser and navigate to http://localhost:5173
